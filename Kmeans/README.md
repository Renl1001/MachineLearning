# Kmeans算法
## 算法流程
1. 加载数据；
2. 随机生成k个点作为初始的质心点；
3. 将每个点的标签选择为与它最近的质心点（本文选用欧氏距离）每个点和最近的质心点结合成了k个簇；
4. 将每个簇的坐标的平均值作为该簇的新质心点
5. 重复3）- 4）直到每个点对应的簇都不发生变化或者达到最大迭代次数
