# KNN算法
## 算法流程
1. 加载数据；
2. 计算测试点与所有训练集的点之间的距离（欧氏距离）；
3. 对距离进行排序，选择与测试点最近的k个点；
4. 计算最近的k个点的标签出现的次数，找到出现次数最多的标签作为测试点的标签。
